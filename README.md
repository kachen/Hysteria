# Hysteria 2 一鍵安裝腳本

在 Linux 伺服器上快速部署 [Hysteria 2](https://v2.hysteria.network/) 代理服務的 Bash 安裝腳本。支援互動式設定、systemd 服務管理、防火牆與 BBR 優化。

## 功能

- 自動下載並安裝最新版 Hysteria 2 二進位檔
- 互動式產生伺服器設定（ACME 自動申請 TLS 憑證）
- 建立 systemd 或 procd 服務並開機自啟
- 自動開放防火牆 UDP 埠（UFW / firewalld / OpenWrt UCI）
- 啟用 BBR 拥塞控制
- 安裝完成後顯示客戶端連線 URI

## 環境需求

| 項目 | 說明 |
|------|------|
| 作業系統 | Linux（一般發行版需 systemd；OpenWrt 使用 procd） |
| 權限 | root（`sudo`） |
| 網域 | 已解析至伺服器公網 IP 的網域名稱 |
| 建議發行版 | Debian 11+、Ubuntu 22.04+、Rocky Linux 8+、CentOS 7+、CentOS Stream 8+、OpenWrt 21.02+ |

不支援：Alpine Linux、NixOS。

## 快速開始

```bash
# 在本機專案目錄執行
sudo bash install.sh

# 或從 GitHub 下載後執行
curl -fsSL -o install.sh https://raw.githubusercontent.com/kachen/Hysteria/main/install.sh
sudo bash install.sh
```

安裝過程中會提示輸入：

1. **網域名稱** — 用於 ACME 申請憑證
2. **信箱** — Let's Encrypt 通知用
3. **監聽埠** — 預設 `443`
4. **偽裝網站 URL** — 預設 `https://www.bing.com/`

密碼會自動產生，安裝完成後會顯示客戶端連線資訊。

## 命令

```bash
sudo bash install.sh install      # 安裝（預設）
sudo bash install.sh upgrade      # 升級至最新版本
sudo bash install.sh uninstall    # 解除安裝
sudo bash install.sh status       # 查看服務狀態
sudo bash install.sh help         # 顯示說明
```

## 環境變數

| 變數 | 預設值 | 說明 |
|------|--------|------|
| `HYSTERIA_USER` | `hysteria` | 執行服務的系統使用者 |
| `LISTEN_PORT` | `443` | 互動式設定時的預設監聽埠 |

範例：

```bash
sudo LISTEN_PORT=8443 bash install.sh
```

## 安裝路徑

| 路徑 | 說明 |
|------|------|
| `/usr/local/bin/hysteria` | 執行檔 |
| `/etc/hysteria/config.yaml` | 伺服器設定檔 |
| `/etc/systemd/system/hysteria-server.service` | systemd 服務單元（一般 Linux） |
| `/etc/init.d/hysteria-server` | procd 服務腳本（OpenWrt） |

## OpenWrt

OpenWrt 路由器可透過 SSH 以 root 執行本腳本。腳本會自動偵測 OpenWrt，並使用 procd 管理服務、UCI 設定防火牆規則。

**前置需求：**

```bash
opkg update
opkg install bash curl ca-bundle openssl-util
```

**安裝：**

```bash
curl -fsSL -o install.sh https://raw.githubusercontent.com/kachen/Hysteria/main/install.sh
bash install.sh
```

**服務管理：**

```bash
/etc/init.d/hysteria-server enable    # 開機自啟
/etc/init.d/hysteria-server start     # 啟動
/etc/init.d/hysteria-server restart   # 重啟
/etc/init.d/hysteria-server status    # 狀態
logread -e hysteria                   # 查看日誌
```

> 注意：OpenWrt 上服務以 root 執行（路由器環境慣例）。MIPS 裝置會自動選用 `mipsle-sf`（軟浮點）二進位檔。

## CentOS 7

CentOS 7 已納入支援。Hysteria 2 官方二進位檔為靜態連結，無需升級 glibc。

腳本會自動偵測舊版 systemd（< 229），並：

- 略過 `AmbientCapabilities` 等新版 systemd 指令
- 透過 `setcap` 授予執行檔網路權限（需 `libcap` 套件）
- 若預設核心不支援 BBR 則跳過（可透過 [elrepo](https://elrepo.org/) 核心啟用）

> 注意：CentOS 7 已於 2024 年 6 月 EOL，建議長期使用請升級至 Rocky Linux 8+ 或 AlmaLinux 8+。

## 服務管理

```bash
# 啟用並啟動
sudo systemctl enable --now hysteria-server.service

# 重啟（修改設定後）
sudo systemctl restart hysteria-server.service

# 查看狀態
sudo systemctl status hysteria-server.service

# 查看日誌
sudo journalctl -u hysteria-server.service -e --no-pager
```

## 編輯設定

```bash
sudo nano /etc/hysteria/config.yaml
sudo systemctl restart hysteria-server.service
```

預設設定檔範例：

```yaml
listen: :443

acme:
  domains:
    - your.domain.net
  email: your@email.com

auth:
  type: password
  password: <auto-generated>

masquerade:
  type: proxy
  proxy:
    url: https://www.bing.com/
    rewriteHost: true
```

更多進階選項請參考 [Hysteria 2 官方文件](https://v2.hysteria.network/docs/getting-started/Server/)。

## 客戶端連線

安裝完成後，腳本會輸出類似以下的連線 URI：

```
hysteria2://<password>@<domain>:<port>/?insecure=0&sni=<domain>#Hysteria2
```

可匯入支援 Hysteria 2 的客戶端使用，例如 [Nekoray](https://github.com/MatsuriDayo/nekoray)、[v2rayN](https://github.com/2dust/v2rayN) 等。

## 解除安裝

```bash
sudo bash install.sh uninstall
```

腳本會詢問是否刪除設定檔目錄（`/etc/hysteria`）與系統使用者（`hysteria`）。

## 常見問題

**服務啟動失敗**

- 確認網域已正確解析至伺服器 IP
- 確認埠號未被其他程式佔用
- 查看日誌：`journalctl -u hysteria-server.service -e --no-pager`

**憑證申請失敗**

- 確認 443/UDP 埠可從外網連入
- 確認防火牆已開放對應 UDP 埠
- 確認網域 DNS 記錄已生效

**重新設定**

```bash
sudo rm /etc/hysteria/config.yaml
sudo bash install.sh install
```
